# NewsReader_Kotlin
news reader application in kotlin.
built using:
  1- MVVM architecture
  2- retrofit
  3- recyclerview with diffutil
  4- nav component
  
 app festurs:
  1- show articles
  2- filter them based on single or multiple selection
  3- details page for articles
  4- favourite articles page
  5- add and remove artilces from favourite
  6- login and signup pages
  7- animated splash screen
